## 1. 游戏概述

新版象棋是一款基于传统象棋、融合战棋与技能系统的双人对战游戏。主要特点：
- 每个棋子拥有**生命值(HP)、攻击力(ATK)、护甲(DEF)**。
- 引入**能量系统**，可释放**微型技能**、**小技能**和**奥义技能**。
- 棋盘扩大至16×12，包含特殊地形。
- 未击杀的移动攻击会触发**堆叠压制**。
- 保留象棋基本移动规则（马脚、炮架、象田等）。

## 2. 棋盘与地形

### 2.1 棋盘尺寸
- **16×12格**，横向A~P，纵向1~12。
- 初始布局：红方在下方（行1-2为军营），黑方在上方（行11-12为军营）。具体坐标与传统象棋相同但向外扩展。

### 2.2 特殊地形

| 地形 | 位置 | 效果 |
|------|------|------|
| 时空裂隙 | 四角A1,A12,P1,P12 | 进入后下回合可瞬移到任意其他裂隙，落地后本回合眩晕（不能移动/攻击/技能）。 |
| 能量洪流 | 中央2×2（H6,I6,H7,I7） | 每回合开始获得双倍能量，但回合结束时受到2点真实伤害。 |
| 祭坛 | 双方底线后方（第0行和第13行），每方5格 | 每方全局一次：献祭一个满能量棋子，为所有友军回复10HP并清除负面状态。 |
| 军营 | 己方最后两行（红1-2，黑11-12） | 每回合额外回复1HP（合计回复3HP）。 |
| 泥沼/疾风 | 由象的奥义创造 | 泥沼：敌方进入消耗2倍移动力；疾风：己方进入移动力+1。 |

## 3. 棋子基础属性

### 3.1 属性表

| 棋子 | 初始HP | ATK | DEF | 移动方式（象棋规则） |
|------|--------|-----|-----|----------------------|
| 帅/将 | 30 | 5 | 0 | 九宫内上下左右1步 |
| 士 | 20 | 6 | 1 | 九宫内斜线1步 |
| 象 | 22 | 6 | 2 | 田（不能塞田） |
| 马 | 18 | 8 | 0 | 日（不能蹩腿） |
| 车 | 25 | 10 | 2 | 横竖任意步 |
| 炮 | 16 | 12 | 0 | 横竖任意步（吃子需炮架） |
| 兵/卒 | 12 | 4 | 0 | 前1步，过河后可左右前；不能后退 |

### 3.2 普通攻击伤害计算
```
伤害 = max(1, 攻击方ATK - 目标DEF)
```
- 真实伤害无视DEF。

### 3.3 生命回复
- 每回合结束时所有棋子回复2HP（军营内额外+1，即3HP）。
- 不能超过初始HP上限。

## 4. 能量系统与技能体系

### 4.1 能量获取
- 移动一格：+10能量
- 普通攻击（造成伤害）：+10能量
- 受到任何伤害：+5能量（每回合每棋子最多触发3次）
- 站在能量洪流上：所有获取的能量翻倍

### 4.2 能量上限
- 默认上限：**100**
- 某些奥义会永久降低上限。

### 4.3 技能分类

| 类型 | 消耗 | 冷却/限制 | 强度 |
|------|------|-----------|------|
| 微型技能 | 10或20 | 每回合每子一次 | 低，辅助 |
| 小技能 | 40~60 | 2~3回合冷却 | 中，战术 |
| 奥义 | 100 | 每方全局2次 | 高，扭转 |

- **每回合每个棋子只能使用一个技能**（微型/小/奥义三选一），不能在同一回合使用多个技能。
- 能量满100时进入“蓄势待发”状态，玩家可选择释放奥义或保留。

## 6. 移动与战斗规则

### 6.1 基本移动
- 遵循传统象棋移动方式（车直、马日、象田等）。
- 马有蹩腿，象有塞田，炮吃子需一个炮架（除非技能解除）。
- 所有棋子不能走出边界。

### 6.2 攻击流程
1. 选择己方棋子和目标格（目标格有敌方棋子）。
2. 计算伤害 = max(1, ATK - DEF)。
3. 显示预览：造成X伤害，目标剩余HP。
4. 玩家确认后执行。
5. **若伤害 ≥ 目标当前HP**：击杀目标，己方棋子移动至目标格。
6. **若伤害 < 目标当前HP**：触发**堆叠压制**（见第7节），己方棋子进入目标格，与敌方共存。

## 7. 堆叠压制机制

### 7.1 触发条件
- 移动攻击未能击杀敌方时，攻击方进入目标格，形成堆叠。
- **压制者**（上方）：攻击方。
- **被压制者**（下方）：被攻击方。

### 7.2 行为限制

| 动作 | 压制者 | 被压制者 |
|------|--------|----------|
| 移动 | 可正常移动离开 | ✅ 可移动离开到相邻空格（离开后堆叠解除） |
| 普通攻击 | 可攻击其他方向敌人 | ❌ 不能攻击压制者；可攻击其他方向，但必须先移动离开 |
| 使用技能 | 正常使用 | 不能使用任何技能（微型、小、奥义均不可） |
| 被外部攻击 | 可被攻击；若死亡，下方上浮 | 可被攻击，但受到伤害时压制者提供50%掩护减伤（向下取整） |
| 范围伤害 | 受全额伤害 | 受全额伤害（无掩护） |

### 7.3 堆叠限制
- 每格最多两个棋子。任何技能尝试导致第三个棋子进入同一格时，第三个棋子滑移到最近空格（若无空格则技能失败）。
- **回合结束时检测**：若己方帅/将在被压制状态，直接判负。

### 7.4 脱离
- 被压制者移动离开后，压制者独自占据该格。
- 压制者移动离开后，被压制者自动上浮成为唯一单位。

### 7.5 技能交互特殊说明
- 炮·超电磁炮击退：若击退目标撞入已有棋子的格子，形成新堆叠（被击退者为压制者）。
- 车·毁灭冲锋：沿途经过堆叠格时，对两个单位均造成5伤害；终点形成堆叠时，车成为新压制者。
- 马·八步赶蝉：被压制者不能使用（需先移动离开）。
- 兵·奋进：若突进后未击杀且进入敌方格，形成堆叠（兵成为压制者）。

## 8. 回合流程

每回合分为三个阶段：

### 8.1 开始阶段
- 所有在能量洪流上的棋子受到2点真实伤害。
- 所有棋子回复HP（2HP，军营内+1）。
- 移除持续一回合的增益/减益（如“激昂”、“震颤”）。

### 8.2 行动阶段
- 当前玩家选择己方一个棋子，执行以下之一：
  - 移动（可附带普通攻击）
  - 使用一个技能（微型/小/奥义，三选一）
- 移动和技能不能在同一回合由同一棋子执行（除非技能描述中包含移动，如“奋进”）。
- 使用技能后消耗能量，进入冷却（如果适用）。
- 移动后若触发攻击，按攻击规则处理。

### 8.3 结束阶段
- 检查胜负条件。
- 切换玩家。

## 9. 特殊规则

### 9.1 士气系统（可选）
- 每击杀一个敌方棋子，全队获得一层士气，每层使所有友军伤害+5%，最多5层。帅/将死亡时移除所有士气。

### 9.2 重伤状态
- 当棋子HP低于30%最大HP时，移动力减半（向下取整，至少1格），但攻击力+2。

### 9.3 召唤与复活
- 士的奥义复活的兵，HP为6（半血），保留过河状态，复活后本回合不可行动。

---

## 10. 胜利条件

满足以下任一条件即可获胜：
1. **击杀将帅**：对方帅/将HP归零。
2. **压制将帅**：对方帅/将在回合结束时处于被压制状态（下方有敌方棋子）。
3. **超时**：竞技模式下每方限时30分钟，超时判负。
4. **认输**。

## 11. 术语表

| 术语 | 解释 |
|------|------|
| HP | 生命值，归零则死亡 |
| ATK | 攻击力，普通攻击基础伤害 |
| DEF | 护甲，减少受到的物理伤害 |
| 真实伤害 | 无视护甲的伤害 |
| 能量 | 使用技能的资源，上限100 |
| 微型技能 | 消耗10/20，每回合每子一次 |
| 小技能 | 消耗40~60，有冷却 |
| 奥义 | 消耗100，每方全局2次 |
| 堆叠 | 两个棋子占据同一格 |
| 压制者 | 堆叠中上方的棋子 |
| 被压制者 | 堆叠中下方的棋子 |
| 激昂 | 下次攻击伤害+50% |
| 震颤 | 下回合无法移动 |
| 弹坑 | 兵自爆留下的地形，进入受2伤/回合 |
| 泥沼 | 移动力消耗2倍 |
| 疾风 | 移动力+1 |
