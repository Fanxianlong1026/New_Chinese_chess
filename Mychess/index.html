玩法速览

棋盘：16×12。红方在下、黑方在上，各自最后两行为军营（每回合多回1HP）。

地形：四角时空裂隙（停留到自己下个回合可瞬移到另一空裂隙）、中央能量洪流（站上去能量翻倍，本方回合末受2真实伤害）。泥沼阻挡敌方进入/穿过，疾风让己方帅士在九宫、兵向前各 +1 步（均由象的奥义生成）。

回血：每个棋子在自己行动方的回合末回复2HP（军营内+1，共3HP），不会在被攻击后立刻回血。

行动：每回合点击一枚己方棋子，执行「移动 / 攻击」或「一个技能」，然后自动换手（双人同屏）。

战斗：伤害 = max(1, 攻击 − 护甲)。若一击未杀死目标，攻击方压在目标上层形成堆叠，下层被压制（不能用技能、只能移动离开）。

能量：移动 +10、造成伤害 +10、受伤 +5（每回合每子至多3次），满100可放奥义。

技能：分微型 / 小 / 奥义三级，每个棋子各 3 个，已全部可用。轻骑、校准、急令、伪装为预备增益，施放后本子可继续移动/攻击；其余技能即为该回合行动。奥义需能量满 100，且每方有全局次数限制。

奥义：末日裁决 / 献祭守护 / 领域改写 / 天翔龙闪 / 末日战车 / 轨道轰炸 / 末日降临 —— 扭转战局的大招，注意全局次数。

胜负：击杀对方将帅；或回合结束时对方将帅处于被压制状态；或认输。

状态点（棋子下方小圆点）：护甲/隐匿 激昂/无敌 震慑 震颤 重伤。
